# `@dagher/quick-hash`

This package uses the built-in Crypto module of Node.js to one-shot hash an
input string with SHA-512, printing it as output.

Usage:

```shell
$ npx @dagher/quick-hash test-string
2fe5d29ee477bb69fd9f8c58e02859b335f6fdaf65a357169e3f946558ca59c6b58ad14caa7243ae4d7074474e89ca6f240e413c067a5419e350cf6d0a1f4a58
```
